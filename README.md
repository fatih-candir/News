## 📖 Proje Hakkında

Bu proje, **React + Vite** kullanılarak geliştirilmiş bir haber uygulamasıdır.  
Farklı haber kaynaklarından verileri çeker, kullanıcıya listeler ve haber detaylarını gösterir.

Bu repo şu anda Vite’in React template’ini temel alıyor ve haber API’si ile genişletilebilir.

---

## 📦 Özellikler

- ⚛️ React + Vite ile hızlı geliştirme
- 📰 Haber listesi gösterimi
- 🔎 Geliştirmeye açık kategori/detay sayfaları
- 📱 Responsive tasarım (isteğe bağlı)
- 🛠 Kolay local geliştirme ortamı
